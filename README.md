#  Smart Water Quality and Water Level Monitoring System

This project is an IoT-based system developed to monitor important water parameters such as **pH**, **temperature**, and **water level** in real time. The sensor data is collected using an ESP32 and sent to a web dashboard through Wi-Fi, allowing users to monitor the water conditions remotely.

## Features

* Real-time pH monitoring
* Real-time temperature monitoring
* Water level monitoring using an ultrasonic sensor
* Live data displayed on a web dashboard
* Wireless communication using Wi-Fi

## Hardware Used

* ESP32
* pH Sensor
* Waterproof Temperature Sensor
* Ultrasonic Sensor (HC-SR04)
* Power Supply
* OLED display

## Software Used

* Arduino IDE
* HTML
* CSS
* JavaScript

## Working

The pH sensor, temperature sensor, and ultrasonic sensor continuously collect data from the water source. The ESP32 reads the sensor values and sends them to the web dashboard over Wi-Fi. The dashboard displays the latest readings, making it easy to monitor the water quality and water level in real time.
